About me

Games industry experience

I have been employed in the games industry continuously since January 2021. I specialise in Unity, having used it in a hobbyist or professional manner since 2013.

  • Raredrop Games - June 2016 - February 2017 (8 months)
  • Ndemic Creations - January 2021 to December 2022 (2 years)
  • Maze Theory - January 2023 to present

Some highlights:

  • Making and releasing my own game, Island Invasion, on Steam in 2018. This was developed in my spare time while working at OVO Energy. I have since taken it down for sale after rolling up the company.
    Steamworks, leaderboards, Unity NavMesh, procedural generation, game architecture
  • Creating the game systems for Warborn.
    Hex-grid, A* pathfinding, unit management, game state management, 2D rendering, Photon networking.
  • Overseeing and releasing Challenge Mode for Rebel Inc (also on mobile).
    Server seeding, "deterministic" random challenges, game state, leaderboards (Google, Steamworks).
  • Co-developing the multiplayer for Rebel Inc on Steam.
    Steamworks Networking, lobby system, network state synchronisation, anti-cheat.
  • Lead games developer at Maze Theory, working on an unannounced VR title.
    Game architecture, AI programming, Github actions CI pipeline, in-house Git expert, mentoring, code review/ standards, planning.

Non-games experience

Professionally I have been a software engineer for over years. In that time I have worked in a number of sectors outside of games, including cyber security, mobile casino development, and energy.

The range of languages and technologies I've used over the years have granted me a quirky and unique set of skills, many of which contribute to my versatility as a games developer.

  • Git - used in all jobs since 2014. Often called upon to solve git woes, be it clearing out refs or solving merge conflicts.
  • CI/CD - managed the CI/CD environment at OVO Energy (CircleCI) and at Maze Theory (Github Actions). Confident in setting up simple ecosystems to boost productivity and build capability.
  • Cloud development - dabbled in GPC at OVO Energy and rolled out some microservices using TypeScript. By no means an expert but know my way around.
  • Web development - you wouldn't know it from this website, but I am competent in raw JS as well as TypeScript.
  • Unix - I've been continuously exposed to Linux systems since I started work in 2012. I know my way around a Unix shell.
